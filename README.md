### Purpose

This project was built as a learning project to practice and understand:

- React fundamentals (components, props, state, hooks)
- React Router DOM for page navigation
- API integration in frontend using Axios
- Context API for state management
- Working with third-party APIs

### About

**Loopa** is a simple e-commerce fashion store where users can browse products, add items to a cart, sign up / log in, and switch between light and dark mode. It uses the [Platzi Fake Store API](https://fakeapi.platzi.com/) for product data and authentication.

### Features

- Browse and search products
- Filter products by category
- View product details with image gallery
- Add to cart, update quantity, remove items
- User sign up and login
- Dark / light mode toggle
- Responsive layout

### Tech Stack

- **React 19** - UI library
- **React Router DOM** - Routing
- **Axios** - API calls
- **Bootstrap 5** - Styling
- **Vite** - Build tool


### Getting Started

```bash
# Clone the repo
git clone https://github.com/Mananpatel08/Loopa-E-commerce.git

# Go into the folder
cd Loopa-E-commerce

# Install dependencies
npm install

# Start the dev server
npm run dev
```

App runs at `http://localhost:5173`

### Scripts
